U.S. Death Trends

This app is built on CDC data that I discovered while searching for alternative data for a regression class at BYU. It is my first attempt at building a shiny application and is primarily an interface for subsetting data and selecting metrics of interest. The user's selected graph is then displayed using the ggplot2 package.
