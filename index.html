Introduction

As I was approaching graduation from my M.S. program in statistics at Brigham Young University I realized that I did not have a good way to share my technical portfolio. I made this website to meet that need so I hope you enjoy exploring!

The rest of the website consists of analysis reports I've written and a Shiny web application that I made to teach myself RShiny. If you find any of the topics, reports, tools or analyses on this website interesting I'm always happy to talk about them! If you have any great resources for me to continue learning from or if you'd like to collaborate on something reach out about that too. :)
